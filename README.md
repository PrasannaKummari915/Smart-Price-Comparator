# 🔥 Smart Price Comparator

A modern web application that helps users **find the best deals across multiple e-commerce platforms** by comparing product prices in real-time.

---

## 🚀 Features

- 🔍 Search products by name
- 🛍️ Compare prices across platforms:
  - Meesho
  - Amazon
  - Flipkart
  - AJIO
  - Myntra
  - Walmart
- 💱 Displays prices in:
  - Indian Rupees (₹)
  - US Dollars ($)
- 🎨 Color-coded platform comparison
- ❤️ Wishlist (saved in browser using localStorage)
- ⚡ Fast and responsive UI
- 🔗 Direct "Buy" links to original platforms

---

---

## ⚙️ How It Works

1. User searches for a product
2. App fetches product data from `products.json`
3. Displays price comparison across platforms
4. User can:
   - View lowest price
   - Click "Buy" to visit platform
   - Add item to wishlist
